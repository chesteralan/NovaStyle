NovaStyle

Visually edit any webpage's CSS — spacing, typography, colors, and layout. Install once, tweak anything.

What is NovaStyle?

NovaStyle is a cross-browser extension that lets developers, designers, and low-code creators modify any webpage's CSS visually. Click elements to inspect and edit their styles in real time through an intuitive control panel, then export clean, production-ready CSS overrides.

📐 Box Model

Drag margins, borders, and padding — see pixel-perfect layout changes in real time.

🔤 Typography

Tweak font-size, line-height, letter-spacing, weight, and alignment with sliders and dropdowns.

🎨 Color & Backgrounds

Interactive color pickers for text and background colors with immediate preview.

💾 Persistent Storage

Styles auto-save per domain and reload on subsequent visits. No data leaves your browser.

📋 Export CSS

Copy formatted CSS override code to your clipboard for use in your project.

🌐 Cross-Browser

Works in Chrome, Firefox, Edge, and Safari with full feature parity.
